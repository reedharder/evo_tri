evo_tri: a python package for the implementation of the Evolutionary Triangulation 
algorithm

version: 1.0
Author: Reed Harder
email: reedharder@dartmouth.edu

Documentation
-----------
Separate documentation coming soon, see code (evo_tri_main.py in particular) for details on function use 

Requirements
-----------
Tested with Python 2.7
Requires installation of numpy and matplotlib (including dependencies) packages

Installation
------------
Simply add evo_tri directory to PYTHONPATH
